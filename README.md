## 🌟 Project Overview

A modern and stylish to-do list web app built with HTML, CSS, and JavaScript.

## ✨ Features
- ✅ Add new tasks
- 📅 Set due dates
- 🔥 Choose task priority
- 🔎 Search tasks
- 🗂️ Filter tasks by All, Active, or Completed
- ✔️ Mark tasks as complete
- 🗑️ Delete tasks
- 🧹 Clear completed tasks
- 💾 Local storage support so tasks remain after refresh

## 📁 Project Structure
```text
Modern To-Do Manager/
```

## ▶️ How to Run
1. Open the project folder in your browser.
2. Open index.html directly, or use a simple local server if preferred.

Example with VS Code Live Server:
- Right-click index.html
- Choose "Open with Live Server"
